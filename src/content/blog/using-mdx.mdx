## 为什么选择 MDX？

MDX 是 Markdown 的一种特殊变体，它支持嵌入 JavaScript 和 JSX 语法。 这解锁了 [将 JavaScript 和 UI 组件混合到你的 Markdown 内容中](https://docs.astro.build/en/guides/markdown-content/#mdx-features) 的能力，用于创建诸如交互式图表或警报之类的内容。

如果你已经有使用 MDX 编写的内容，那么这个集成有望使迁移到 Astro 变得轻而易举。
